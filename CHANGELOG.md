# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/) and this project adheres to
[Semantic Versioning](http://semver.org/).

## [Unreleased]

## [v1.4.0] - 2026-04-29

### Added
- Allow utilisation of llms-txt-php `v3.*`.

### Fixed
- New `curl` extension availabilty guard. Closes [#2](https://github.com/raphaelstolt/llms-txt-php/issues/2).

## [v1.3.0] - 2026-04-29

### Removed
- Drop PHP `8.1` support.

## [v1.2.0] - 2026-04-22

### Added
- New `render` command to render the llms.txt file.

## [v1.1.0] - 2026-01-30

### Added
- Four new AI skills for `validating`, `checking-links`, `infoing`, and `initialising` a llms.txt file.

## v1.0.0 - 2025-07-29

- Initial release.
